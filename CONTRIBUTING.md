# Contributing

The Threshold is maintained as an evidence archive first and a publication second.

## Before adding a claim

1. Identify the original claimant and date.
2. Prefer the complete primary source.
3. Determine whether later articles independently confirm the claim or repeat one chain.
4. Add the source to `src/_data/sources.json`.
5. Add or update a record in `src/_data/timeline.json`.
6. Classify verification status separately from confidence.
7. Run `npm run validate`.

Do not turn visits, follows, likes, colors, jokes, or suggestive imagery into project confirmation.

## Corrections

Do not silently revise a published edition. Add a dated correction object, update underlying research data, and add a changelog entry.

## Style

Use exact dates, neutral wording, semantic HTML, locally available system fonts, and progressive enhancement. Preserve readability over decorative effects.
